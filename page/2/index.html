实习生面试-字节跳动

发表于 2019-12-13 更新于 2020-02-10

岗位：后端研发

一面

  1. 代码 两个有序数组，找中位数
    双指针O(n),二分套二分O(logn ^ 2)

  2. 代码 给字母矩阵(9 * 9),问是否存在一条路径构成询问的单词(如果有多个询问单词呢)

  3. mysql中主键和索引有什么区别

    • 主键一定是唯一性的索引，唯一性的索引不一定是主键
    • 唯一性索引可以有多个，主键只能有一个
    • 主键列不允许为空值，唯一性索引可以为空值
    • 主键列可以由多个字段组成，形成复合主键，同时主键也是唯一索引
    • 唯一索引表示索引值唯一，可以由一个或多个字段组成，一个表可以有多个唯一性索引

  4. TCP四次挥手(收到了FIN的ack,但是对面的FIN迟迟不来怎么办)

  5. 线程和进程的区别

二面

  1. python的多线程是假的
    python的代码的执行由python虚拟机来控制，同时只有一个线程在执行
    解决方法
    • 重写python编译器(pypy)
    • 调用c语言链接库
  2. 64匹马,8个赛道,最少比几次找到最快的4匹
  3. 代码 实现memcpy
  4. 代码 共有n列,m个方块,每块1*1*1,落在第a[i]列,消掉一行得1分,问最后得分
  5. TCP三次握手四次挥手,同时关闭
  6. TCP连接为什么可靠
  7. 线程和进程的区别
  8. 进程调度算法,线程调度
  9. linux命令 pwd,电脑卡的时候用什么命令
  10. 一枚不均匀的硬币,怎么制造两个等概率事件
  11. 内存结构 缺页置换

三面

  1. 上次面试看你有些不会,回去有学习吗,学了些什么
  2. 代码 给定一个二叉树以及树上的部分节点,如何找到这些节点的最近公共祖先
    lca O(nlogn) && 树形dp O(n)
  3. 代码 求二叉树的直径
  4. 巴什博弈
  5. 有一个[0,4]的随机数生成器,怎么生成[0,6]的随机数
  6. 一个省的高考成绩怎么排序
    直接排序O(nlogn),桶排O(n)

总结

牛客投的杭州,被推到了北京,说给我转到上海,后来不想放我走,面试又问我去不去深圳..
总体体验不错,面试效率相当高,面试问题比较有深度,有大公司的风范

实习生面试-商汤

发表于 2019-12-11 更新于 2020-02-10

岗位：自动驾驶

一面

简单面了一下，给我安排到其他组

  1. 代码 快排
  2. 代码 最长公共子序列

二面

  1. 给n个点的完全图，点编号0~n-1，边权为两个顶点的异或和，求最小生成树。(n<=2^32)
  2. 给一个序列，长度n(n<2e6)。问当k分别为1-n时，原串中长度为k的连续子串中的最小值的最大值。
    用单调队列预处理出每个数之前第一个比他小的位置和之后第一个比他小的位置，可以得到这个数能影响的长度len,ans[len] = max(ans[len],a[i]) 然后倒着扫一遍即可。
  3. 知道智能指针吗
  • ptr->func()?
  • 代码有什么问题？class内的public和继承的public有什么区别？

三面

  1. 定义class只能在堆栈中使用
  2. n个台阶，每次最多能走n个，问走到n的方案数
  3. 一个稍微动点脑子的小学数学题

总结
